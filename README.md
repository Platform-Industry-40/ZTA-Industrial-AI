## Teaser

This repository documents the development and implementation of a **Zero-Trust Architecture (ZTA)** specifically tailored for industrial AI applications.  
It addresses the context of Industry 4.0, the EU AI Act, geopolitical boundary conditions, and practical Proof-of-Concept implementations for selected use cases.

The goal is to enable trustworthy, secure, and auditable AI systems in manufacturing companies.
